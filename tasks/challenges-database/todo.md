## Implementation

- Status: `[done]`
- Goal: AI-generated daily challenges. Each day, for each language+difficulty combo, a fresh challenge is generated by Claude AI on first request and cached for the rest of the day.
- Why: Static pre-seeded challenges are repetitive. AI-generated challenges provide unique daily content.
- Inputs: User selects language + difficulty ‚Üí AI generates 5 quiz questions ‚Üí user answers ‚Üí results saved.
- Output: Challenges fetched from DB (cached daily) or generated fresh by AI. Attempt results and XP tracked per user.
- Target files:
  - `src/lib/db/schema.ts` ‚Äî languages, categories, tags, challenge_tags, challenges, questions, challenge_attempts, user_stats tables
  - `src/lib/ai-challenge.ts` ‚Äî Claude AI challenge generation service
  - `src/app/api/challenges/route.ts` ‚Äî GET challenge (cached daily or AI-generated)
  - `src/app/api/challenges/[id]/submit/route.ts` ‚Äî POST submit answers
  - `src/app/api/challenges/today/route.ts` ‚Äî GET today's completed attempts
  - `src/app/api/user/stats/route.ts` ‚Äî GET user stats
  - `src/app/api/languages/route.ts` ‚Äî GET available languages from DB
  - `src/app/(frontend)/challenges/page.tsx` ‚Äî fetches from API, languages from DB
  - `scripts/seed-languages.ts` ‚Äî seed languages and categories
- Acceptance criteria:
  - [x] Schema: `languages`, `categories`, `tags`, `challenge_tags` tables with proper FKs
  - [x] Schema: `challenges` references `language_id` and `category_id`, has `generated_date` for daily caching
  - [x] AI generation: Claude Haiku generates 5 questions per challenge
  - [x] Daily caching: same language+difficulty on same day returns cached challenge
  - [x] API: GET `/api/challenges?language=X&difficulty=Y` returns AI-generated challenge
  - [x] API: POST `/api/challenges/[id]/submit` saves attempt and awards XP
  - [x] API: GET `/api/user/stats` returns user's XP, level, streak, rank
  - [x] API: GET `/api/languages` returns available languages from DB
  - [x] Challenges page fetches languages from DB, not hardcoded
  - [x] 5 languages + 10 categories seeded via `scripts/seed-languages.ts`
- Validation:
  - [x] `bun run build` passes successfully
  - [ ] manual check: register ‚Üí pick challenge ‚Üí AI generates ‚Üí answer ‚Üí submit ‚Üí XP updates ‚Üí next day gets new challenge
- Notes:
